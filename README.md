#React Boilerplate made with :heart: and :coffee:

##Getting Started :bulb:

#### 1. Install `nodejs if you dont have it: https://nodejs.org/en/`
#### 2. clone this repository `git clone`
#### 3. cd into the repository `cd react-boilerplate`
#### 4. Run `npm install`
#### 5. Run `npm start`
